# PIZZA PALACE
#### 25.05.2020
#### By **IAN AMUGA**
## Description
Pizza palace is a web application that enables a user to chose a pizza, toppings and the crust desired, select the quantity and order it online.
## Setup/Installation Requirements
No setup required, just access the site and make the order
